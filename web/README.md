# shopSmsReserv (web)

## 開発環境セットアップ

このリポジトリは **Next.js (React) + Tailwind CSS + TypeScript** で構築します。

### 前提

- Node.js 18+（推奨: 20+）

### インストール

```bash
cd web
npm install
```

### 起動

```bash
npm run dev
```

起動後、以下にアクセスします：

- 顧客向け（仮）: `http://localhost:3000/`
- 管理画面: `http://localhost:3000/admin`

## 管理画面の使い方（モック）

現時点の管理画面は **UI/UX 確認用のモック**です（Supabase 未接続のため、DB 保存や実 SMS 送信は行いません）。

### アクセス（IP 制限：モック）

- **初回アクセス時**、`/admin` 配下に入る前に「現在のアクセス元 IP（仮）」入力が表示されます。
- **許可 IP リスト（ホワイトリスト）方式**です。
  - **追加した IP = 許可**
  - **リストから削除した IP = 拒否**
- 許可 IP の管理は `http://localhost:3000/admin/access-logs`（アクセス許可とログ）から行えます。

### 画面（タブ）とできること

- **本日の予約一覧と予約受付**（`/admin`）
  - 当月カレンダーで日付選択
  - 右側の一覧で「空き」をクリック → **予約作成（モーダル）**
  - 「予約あり」をクリック → **予約詳細（モーダル）**
    - **変更 / キャンセル / SMS 再送（仮）**
- **営業時間と日別営業時間の設定**（`/admin/settings`）
  - デフォルト営業時間の設定
  - 昼休憩（デフォルト）の ON/OFF と開始/終了
  - 営業日（週：月曜〜日曜）の設定
  - 日毎の営業時間（個別）と、昼休憩（個別）の上書き
  - 予約キャンセル・変更期限（1〜48 時間前）
- **施術メニュー編集**（`/admin/treatments`）
  - 施術メニュー（概要/施術時間/価格）の編集 UI（ダミー表示）
- **アクセス許可とログ**（`/admin/access-logs`）
  - 許可 IP の追加/削除（追加＝許可、削除＝拒否）
  - アクセスログ記録 ON/OFF（モック）
  - アクセスログ一覧（許可/拒否、ページ）
