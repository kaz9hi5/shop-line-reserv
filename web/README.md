# shopSmsReserv (web)

## 開発環境セットアップ

このリポジトリは **Next.js (React) + Tailwind CSS + TypeScript** で構築します。

### 前提

- Node.js 18+（推奨: 20+）

### インストール

```bash
cd web
npm install
```

### 起動

```bash
npm run dev
```

起動後、以下にアクセスします：

- 顧客向け（仮）: `http://localhost:3000/`
- 管理画面: `http://localhost:3000/admin`
